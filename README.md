# demand_forecasting
I built a full end‑to‑end Demand Forecasting Machine Learning Platform that lets users explore data, train models, track experiments, manage a model registry, and run real‑time predictions — all through a clean, intuitive web interface.

This project demonstrates my ability to design and implement a complete ML workflow, combining data engineering, model development, backend architecture, and frontend visualization.

🔍 What the Platform Does
The system supports the entire machine learning lifecycle:

1. Dataset Exploration
Loads a 76,000‑row retail dataset

Displays a clean preview table

Shows metadata (columns, missing values, feature list)

Ensures data quality before training

2. Model Training
Supports multiple regression models

Computes MAE and RMSE

Stores:

Model file

Metrics

Timestamp

Feature list

Provides real‑time success feedback

3. Experiment Tracking
Logs every training run

Displays results in a sortable table

Highlights best‑performing models

Functions like a lightweight MLflow

4. Model Registry
Centralized storage for all trained models

Shows metrics, feature count, and file paths

Enables model selection for inference

5. Inference Dashboard
Predicts demand for any product

Displays item details

Renders a prediction history chart

Smooth, business‑style UI

🧱 Tech Stack
Backend: Flask, Python, SQLite
ML: scikit‑learn, Pandas, Joblib
Frontend: HTML, Bootstrap, Chart.js
Architecture: Modular, extensible, production‑style layout
